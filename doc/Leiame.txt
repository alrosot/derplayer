DerPlayer 1.0
12/02/2008

=============================================

INSTALAÇÃO

Para instalar o Der Player, basta descompactar o arquivo
"DerPlayer.jar" e o diretório "lib" em um mesmo diretó-
rio do seu sistema.

Atenção! É recomendado que o diretório possua permissão
de escrita (este aviso serve principalmente pra galera 
do Linux). Mas caso o player seja colocado em algum lugar
onde o mesmo não tenha permissão para criar arquivos fun-
cionará da mesma forma, mas com uma queda de desempenho.


EXECUÇÃO

Para rodar o programa, é preciso ter o Java versão 6 ou supe-
rior instalado em seu sistema. Caso você não o possua, este
pode ser obtido em http://www.sun.com/getjava

Em um ambiente Windows, geralmente basta dar um clique duplo
no arquivo DerPlayer.jar para iniciar a execução.

Caso isso não funcione, você pode executá-lo digitando o se-
guinte comando na pasta em que o player foi descompactado:
java -jar DerPlayer.jar

Dica: outra alternativa aos usuários Windows (quando o clique
duplo não funciona) é clicar no DerPlayer.jar com o botão
direito, ir em "Abrir com" ("Open with") e escolher
Java(TM) Platform SE Binary



UTILIZAÇÃO

O DerPlayer possui apenas as funções básicas de um player.
O único segredo que vale ser ensinado aqui é a localização
do "Próxima música".

Como esta é provavelmente a função mais utilizada de um
player, ela foi colocada no lugar de maior acessibilidade:
basta dar um clique simples no ícone do DerPlayer!
